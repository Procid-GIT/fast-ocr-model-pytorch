# OCR Model
Simple OCR for recognizing just individual characters as of now, that I made.

_______________________________
This was a project I made, for fun, mainly because I wanted to practice AI, and learn more Neural networks, and also contribute to my Github.  <br>
<p>        <br></p>
The AI can recognize sentences, though its quite inacurrate and early in testing.  It uses a Convolutional Neural Network(CNN), followed by a dense MLP in order to analyze and process them.  A softmax function is then applied to output its classifications.  <br>
