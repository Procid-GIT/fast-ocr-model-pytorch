This was a project I made, for fun, mainly because I wanted to practice AI, and learn more Neural networks, and also contribute to my Github.  <br>
<p>        <br></p>
The AI can only recognize numbers from 0 to 9, although I eventually plan to make it recognize letters, and then recognize full sentences and paragraphs like this.  
It uses a Convolutional Neural Network(CNN), followed by a dense MLP in order to analyze and process them.  A softmax function is then applied to output its classifications.  <br>
